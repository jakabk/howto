#http
#status
#code

- 1×× - informational
- 2×× - success
    - 201 - Created
- 3×× - redirection
    - 303 - See Other
    - 307 - Temporary Redirect
    - 308 - Permanent Redirect

- 4×× - client error
    - 400 - Bad Request
    - 401 - Unauthorized
    - 402 - Payment Required
    - 403 - Forbidden
    - 404 - Not Found
    - 409 - Conflict (if resource to create already exists)

- 5×× - server error
    - 500 - Internal Server Error
    - 501 - Not Implemented
    - 502 - Bad Gateway
    - 503 - Service Unavailable
    - 504 - Gateway Timeout
